### [6. uat](./6-uat/)

The purpose of this stage is to set up the environment projects which contain the resources for individual business units.

This will create the following project structure:

```
example-organization
└── fldr-dev
    └── prj-zzzz-d-app1
└── fldr-prod
    └── prj-zzzz-p-app1
└── fldr-qa
    └── prj-zzzz-q-app1
└── fldr-uat
    └── prj-zzzz-u-app1
```
